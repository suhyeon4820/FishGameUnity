# FishGameUnity

- 게임 시스템

  - **UIFishGame** : 낚시 성공 시 잡은 아이템 이름/포인트를 나타내고 총 미끼 수와, 포인트 정보를 나타냄 

  - **FishGameManager** : FishItem에서 게임 아이템 정보를 가져와 게임 상태 전환 및 게임을 운영하고 게임 정보를  UIFishGame에 전달

  - **FishItem** : 게임 아이템 오브젝트의 정보(종류, 획득 포인트, 획득 확률)를 다룸
